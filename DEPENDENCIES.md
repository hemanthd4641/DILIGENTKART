# Project Dependencies

This project uses two main package.json files for managing dependencies:

## Backend Dependencies (ecommerce-app/backend/package.json)

- express: Web framework for Node.js
- mongoose: MongoDB object modeling tool
- dotenv: Environment variable loader
- cors: Cross-Origin Resource Sharing middleware

## Frontend Dependencies (ecommerce-app/frontend/package.json)

- react: JavaScript library for building user interfaces
- react-dom: Entry point for working with the DOM
- react-router-dom: Routing library for React applications
- axios: Promise based HTTP client
- tailwindcss: Utility-first CSS framework
- express: Web framework (for serving frontend)
- http-proxy-middleware: Proxy middleware for API requests

All dependencies can be installed by running `npm install` in their respective directories.
